# Project Memory

## Core
Obsidian Forest dark theme is the default everywhere (public + app). Light mode + custom palettes opt-in via Settings. Brand: sage `#8AB79B`, emerald bloom `#2D4A3E`, obsidian bg `#0D1312`.
App uses `UserPrefsProvider` — never set `theme-dark`/`theme-light` body classes manually inside `/app`; let the provider drive it. Auth pages (login/signup) explicitly set `theme-dark`.
Logo: `src/assets/chronosquares-logo.png` rendered via `CSLogo` component. Wordmark "Chrono**Squares**" with brand-coloured "Squares".
